Method 1: Using FreeDVD Boot

Necessary Downloads

Method 2: Weird Install Using FreeHardDriveBoot (for FreeMcBoot users looking to transition)

Necessary Downloads

  • FreeHardDriveBoot
  • HDD Raw Copy Tool
  • PSBBL
  • PSBBL+Open Tuna
  • Free Mcboot Installer
  • Custom SYS-Comf file

Instructions

  1. Select 'FHDB v1.966 (2019-01-14).img' as source, click continue, select PS2 HDD as target, click continue and click start.
  2. Place PS2 HDD into PS2 using network adapter.
  3. Put PSBBL, PSBBL+Open Tuna, and Free Mcboot installer onto a Fat32 formatted USB Drive and insert it into your PS2.
  4. Turn on your PS2 and launch wLaunchELF.
  5. Backup your memory card: Navigate to your target memory card, select all files using X, and copy using R1. PSU paste into the Mass.
  6. Navigate to FreeMcBoot installer ELF in Mass and click the circle button to run.
  7. Press R1 twice until you see a prompt to format your memory card. Format the memory card and exit.
  8. Open PSBBL KELF file and install using a normal install.
  9. If after installing PSBBL it does not go into OPL, press R1 during boot until you get into wLaunchELF to try and launch XEB+.
  10. If wLaunchELF does not launch, remove memory card, boot into FreeHDD Boot, insert memory card, launch wLaunchELF, open PSBBL+Open Tuna ELF and install.
  11. During boot press R1 to go into wLaunchELF and copy the SYS-CONF file from Mass to the memory card you installed PSBBL.

XEB+ Setup

  1. Download the Xtreme Elite Boot Plus Xmas 2021 showcase here.
  2. Extract the XEBPLUS folder to the root of your USB drive, and ensure that you can load into the XEB+ dashboard on your PlayStation 2.
  3. Note that if you are using an exFAT formatted USB drive, you will need to use PSBBL and this version of wLaunchELF.
